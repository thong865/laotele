## Installing

### Package manager

Using npm:

```bash
$ npm install lao-tele
```

Using bower:

```bash
$ bower install lao-tele
```

Using yarn:

```bash
$ yarn add lao-tele
```

Using pnpm:

```bash
$ pnpm add lao-tele
```


```js
import laotele from 'lao-tele';

console.log(laotele('55555555'));
````
